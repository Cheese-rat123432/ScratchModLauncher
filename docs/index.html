Setup a mod

Setup a mod from Discover tab

Choose a mod, rightclick it and click "Setup a directory for mod"

Note: Some of the mods listed in Discover tab does not link to site with download, that means some mods cannot be downloaded



for .image file

Click Open in -> Squeak



for .swf file

Note: You need to install Newgrounds player!

Click Open in -> NG player,
click .swf name, type in .swf name and press enter



for .exe and .lnk file

Click Open in -> Custom,
click .exe name and enter file name (with extension)



Finish

Click done, then press esc or close Friendly mod setup for SML in other way,
open created mod directory,
paste your mod in



Setup a mod manualy

Click on folder icon next to Settings tab,


open mods folder,


rightclick > New > Folder, name it with your mod name, open that folder,


rightclick > New > Text document, open it,


Write:

name="[name]"
defold="[folder in SML]"

replace [name] with mod name
replace [folder in SML] with any folder name (so your Launch tab will be organised)


save the text file and rename it to new.lua

Note: you need to have enabled show file extension,
in file explorer click on View tab and in Show/hide section enable File name extensions, detailed instructions here

then go to SML root folder, open modsetup.exe and follow instructions here
